# Proyectos Varios

**Colección de proyectos independientes de programación y lógica aplicada — desde visualización en consola hasta criptografía y juegos. Cada uno explora una idea concreta con implementaciones limpias y evolutivas.**

---

## Proyectos

| Carpeta | Tema | Descripción |
|---------|------|-------------|
| [`Display_LED/`](Display_LED/) | Visualización ASCII | Renderiza números enteros como display LED de 7 segmentos en consola, con tres versiones que evolucionan hacia un diseño funcional. |
| [`Encriptador_Cesar/`](Encriptador_Cesar/) | Criptografía clásica | Cifrado César extendido a 99 caracteres ASCII (letras, dígitos, puntuación, espacios) con encriptación y desencriptación. |
| [`Tricky/`](Tricky/) | Juego de lógica | Tres en línea jugable en consola contra la máquina, con tablero ASCII, turnos recursivos y detección de victoria/empate. |

Cada proyecto tiene su propio `README.md` con documentación detallada.
